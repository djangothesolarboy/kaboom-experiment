kaboom-experiment
---
This is an experiment using the [kaboom.js](https://kaboomjs.com/) game library.

*Note: The game **does** have sound.*

Play the game [here](https://blue-boy.netlify.app/).  

Controls:
---
- Menu: 
    - Press **Space** to start game.
- Game: 
    - **WASD** || **ARROWS** for movement.
    - **R** restarts game.
    - **SPACE** || **W** || **UP** to jump.
